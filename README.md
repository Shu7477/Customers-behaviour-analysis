# 📊 Customer Behavior Dashboard (Power BI)

An interactive **Power BI dashboard** built to analyze customer behavior and purchasing patterns. This project provides insights into customer demographics, purchase behavior, subscription status, geographical distribution, and revenue trends through interactive visualizations.

## 📌 Project Overview

The **Customer Behavior Dashboard** helps businesses better understand their customers by analyzing demographic information, purchasing habits, customer reviews, subscription status, and revenue distribution. The dashboard enables quick decision-making through an intuitive and interactive interface.

---

## 📈 Key Insights

- 👥 Analyze customer distribution across different **age groups**.
- 💰 Monitor the **average purchase amount**.
- ⭐ Track the **average customer review rating**.
- 📍 Explore customer distribution across different **US states**.
- 🛍 Compare purchase amounts based on **subscription status**.
- 🚻 Visualize **revenue contribution by gender**.
- 📅 Understand customer purchasing frequency over time.
